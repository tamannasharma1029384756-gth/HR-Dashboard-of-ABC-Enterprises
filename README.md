Project Overview
This project presents an HR Analytics Dashboard created to analyze and visualize workforce data of ABC Enterprises. The dashboard provides insights into employee demographics, salary distribution, education levels, workforce growth, and leave balances.  
The objective of this dashboard is to help HR teams and management understand employee trends, make informed decisions, and monitor workforce development over time. The dashboard was designed to provide a clear and interactive visualization of HR metrics, allowing users to quickly identify patterns such as gender distribution, age groups, salary comparisons, and job role distribution.
Key Features of the Dashboard
1. Employee Distribution by Job Title
2. Gender Distribution
3. Age Distribution of Employees
4. Workforce Growth Over Time
5. Salary Insights
6. Education vs Salary Analysis
7. Leave Balance Analysis
These insights can help HR teams improve workforce planning, compensation strategies, and talent development initiatives.
